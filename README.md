### 쿠키(Cookie)
#### 서버에 접속시 접속한 클라이언트의 정보를 클라이언트에 저장
#### 이후에 서버로 전송되는 요청에는 쿠키가 가지고 있는 정보가 같이 포함되어서 전송
#### 쿠키는 4kb이하
#### 쿠키는 이름, 값, 유효기간, 도메인, 경로 등으로 구성

#### 사용 예, 로그인 시 아이디 저장, 최근 검색 키워드등
